# Formulary

```html
<form>
<input/>
- Id
- Type = text, checkbox, radio (use the same prop name for all of options),submit
<textarea>
 - cols
 - rows
<label>
- for  // id of input. Ex for-“message”

<select>
	<option>text</option>
	<option>text</option>
	<option>text</option>
</select>

<fieldset> //join some inputs like checkboxes
	<legend> // label
</fieldset>

```
